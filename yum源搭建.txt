Create Local Repos
参考文档：https://wiki.centos.org/HowTos/CreateLocalRepos
创建rpm包存放目录:
[root@server1 ~]# mkdir -p /yum/centos
如果createrepo 没有安装,则需要安装.
[root@server1 ~]# yum install createrepo
放入RPM包并创建 metadata:
[root@server1 ~]# mv /$home/RPMS/* /yum/centos
[root@server1 ~]# chown -R root.root /yum/centos 
[root@server1 ~]# createrepo /yum/centos
[root@server1 ~]# chmod -R o-w+r /yum/centos
注意: 当有新的安装包放入时请重新执行第三和第四步
创建 /etc/yum.repos.d/local.repo
[local]
name=CentOS-7
baseurl= http://10.95.218.120/centos/
enabled=1
gpgcheck=0

测试yum源.
[root@server1 ~]# yum install foo foo-devel foo-docs
登陆到客户端配置yum信息.
[root@client1 ~]# scp server1:/etc/yum.repos.d/local.repo /etc/yum.repos.d/
测试yum源.
[root@server1 ~]# yum install foo foo-devel foo-docs
